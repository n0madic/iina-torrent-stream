How much data the engine keeps queued ahead of the play cursor — a larger window keeps more peers busy on a slow swarm. Clamped internally to 256–2048 MiB. Does not cap on-disk usage; downloaded data lives in a temporary folder that is removed when the engine stops. Default: 512 MiB; raise it if playback stalls on a slow swarm.

The mpv demuxer buffer (demuxer-max-bytes). Larger values smooth playback over slow swarms at the cost of memory. Default: 256 MiB.

Upload pieces back to the swarm while the engine is running. Disabling this is less polite to the torrent community.

Reveal the Torrent sidebar (download progress, peers, file list) when a torrent starts playing.

Exposes /debug/memstats (live heap / goroutine / torrent counts) and the standard Go /debug/pprof/* profiler endpoints on the daemon's localhost HTTP server. Use to diagnose memory or performance issues. Off by default. Restart IINA after toggling — the daemon reads this flag at startup.
